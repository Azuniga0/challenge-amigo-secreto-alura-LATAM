# Qué es el juego del amigo secreto
Es una aplicación web que implementa lógica de programación en JavaScript aprendida en los cursos anteriores de alura LATAM G9. 


## Versiones
Esta es la primer entrega de la aplicación, con posibles mejoras a futuro de acuerdo a los requerimientos provistos.


## Requisitos
Navegador web de su preferencia actualizado.


## Funciones
Dentro de las funciones se encuentran
- Ingresar nombres de personas.
- Validar que no intente agregar campos vacíos al arreglo con una alerta.
- Validar que no ingrese nombres duplicados.
- Validar que los nombres cumplan con el formato de solo letras del abecedario, vocales acentuadas, espacios y caracteres especial como la 'ñ' o 'Ñ'.
- Agregar los nombres validados a un arreglo de nombres para futuras operaciones dentro de la aplicación.
- Mostrar un mensaje de advertencia si el nombre no cumple con el formato o si fue agregado exitosamente.
- Mostrar los nombres que van registrando en una lista html debajo del recuadro para escribir recorriendo el arreglo con el ciclo for.
- Verificar que existan al menos dos nombres ingresados para que pueda llevarse a cabo el sorteo.
- Generar un número aleatorio entero no repetible de acuerdo a la cantidad total de participantes para identificar al ganador y asegurar que todos los registrados tengan una oportunidad de ser sorteados.
- Mostrar el nombre del afortunado ganador cada que presione el botón 'Sortear' y aun queden nombres disponibles.
- Mostrar un mensaje de que todos los participantes han sido sorteados y deshabilitar el botón de sortear cuando ya no queden nombres disponibles.
